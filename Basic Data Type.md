######变量在内存中的表示：
eg：a = 'ABC'

执行这条语句时时，Python解释器干了两件事情：在内存中创建了一个'ABC'的字符串；在内存中创建了一个名为a的变量，并把它指向'ABC'。

也可以把一个变量a赋值给另一个变量b，这个操作实际上是把变量b指向变量a所指向的数据，例如下面的代码：

a = 'ABC'   **解释器创建了字符串'ABC'和变量a，并把a指向'ABC'：**

b = a       **创建变量b，并指向a**

a = 'XYZ'   **创建字符串'XYZ',且使a指向'XYZ'**

print(b)    **此时b还指向'ABC'**
